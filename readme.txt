This is a README on how to use various IDL scripts in this folder

ExamineMap looks at weather -,+ 6 hours(left and right column) with rows corresponding to pressure levels
	EG: ExamineMap(jday=julday(3,31,2010,0),/melbourne)

windmap looks at wind and GPH at only one pressure and one jday
	EG: windmap(julday(2,3,2005,0),/melbourne,/highres)
